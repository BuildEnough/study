# 문자 중간에 변수 넣기
```js
console.log(1 + '2'); // 12
```
- JavaScript에서 `+`기호는 문자 이어 붙이기가 가능함
- `숫자 + 숫자`: 숫자 뎃셈
- `문자 + 숫자`: 문자 덧셈(1개가 문자면 문자 덧셈)

<br>

```js
var a = '안녕';
console.log('문자' + a + '문자'); // 문자안녕문자
```
- 문자 중간에 변수 넣고 싶으면 `+`기호 넣으면 됨

<br>

```js
var a = '안녕';
console.log(`문자 ${a} 문자`); // 문자안녕문자
```
- (`)백틱 기호 사용해도 됨
    1. 문자 중간에 엔터키 칠 수 있음
    2. 중간에 `${변수명}` 문법 이용가능
