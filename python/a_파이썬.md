# 파이썬
인터프리터 언어  
- 소스 코드를 한 줄씩 해석한 후 실행하여 결과를 바로 확인할 수 있는 언어

## 파이썬의 특징
1. 인간다운 언어
2. 문법이 쉬어 빠르게 배울 수 있다
    - 프로그래밍 경험이 있다면 일주일이면 충분하고 하긴함
3. 무료이지만 강력하다
    - 뼈대는 `파이썬`으로 만들고 빠른 실행속도가 필요한 것은 `C`로 만들면 된다
4. 간결하다
5. 개발속도가 빠르다

## 파이썬으로 할 수 있는 일
1. 웹 프로그래밍
2. 인공지능과 머신러닝
3. 수치 연산 프로그래밍
    - `Numpy`
4. 데이터 분석
    - `pandas`
5. 데이터베이스 프로그래밍
    - `pickle`
6. 시스템 유틸리티 제작
7. GUI 프로그래밍
    - `Tkinter`
8. 사물 인터넷
    - `라즈베리 파이`
