## 2. 함수 스코프 vs 블록 스코프
- 함수 안에서 선언된 변수는 함수 밖에서 접근할 수 없다
    ```python
    def my_function():
        func_var = "함수 안의 변수"

    my_function()
    # print(func_var)  # 오류! 함수 밖에서는 접근 불가
    ```
- if, for, while, with 등의 블록들 안에서 선언된 변수는 블록 밖에서도 접근할 수 있다
    ```python
    for x in range(3):
        print(x)
    print(x) # 2
    ```

## 3. 프로그램의 입출력
```python
# 대문자 변환 프로그램
import sys
args = sys.argv[1:] # 처음은 자기자신이 나와서 다음꺼부터
for i in args:
    print(i.upper(), end =" ")
```
