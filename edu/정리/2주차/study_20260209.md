- 참조
```python
e = [11, 22, 33]
ee = e
ee[2] = 77
print(ee is e, ee, e) # True [11, 22, 77] [11, 22, 77]
```
- 복사
```python
e = [11, 22, 33]
ee = e[:]
ee[2] = 77
print(ee is e, ee, e) # False [11, 22, 77] [11, 22, 33]
```
- copy() 함수를 사용한 복사
```python
e = [11, 22, 33]
ee = e.copy()
ee[2] = 77
print(ee is e, ee, e) # False [11, 22, 77] [11, 22, 33]
```

- 두 변수의 값을 바꿀 때
```python
a = 33
b = 55
print(a, b)

a, b = b, a
print(a, b)
```
